# Tool Design Rules

## Core UX Rules
- Tools must load fast and work without ads
- No gating, forced interactions, or paywalls
- Results must be clear and actionable

## Ad Placement for Tools
- Ads may appear only after results are shown
- Never place ads before inputs or results
- Ads must not block interaction or visibility

## Privacy and Data
- Favor local, in-browser calculations
- Do not collect unnecessary personal data
- Be transparent about any data usage

## Accessibility
- Keyboard and screen-reader friendly
- Clear labels and error states
- Mobile-first layout

## Safety and Compliance
- No legal advice or guarantees
- Use disclaimers where needed
- UX and trust override monetization
