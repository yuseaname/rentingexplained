# RentingExplained.com - Deployment & Next Steps

## ?? Immediate Next Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Visit http://localhost:3000 to see the site locally.

### 3. Test All Features
- ? Navigate to all pages
- ? Test interactive tools (Budget Checker, Fees Estimator, Lease Scanner)
- ? Complete a quiz
- ? Check leaderboard and badges
- ? Test reading progress and TOC
- ? Verify all 5 articles render correctly

### 4. Deploy to Vercel (Recommended)

**One-Click Deploy:**
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Follow prompts to connect GitHub and deploy
```

**Or use Vercel Dashboard:**
1. Go to https://vercel.com
2. Click "New Project"
3. Import your GitHub repository
4. Vercel auto-detects Next.js - click "Deploy"
5. Your site will be live in ~2 minutes

### 5. Post-Deployment Configuration

**Google Search Console:**
1. Add property at https://search.google.com/search-console
2. Verify ownership
3. Submit sitemap: `https://yourdomain.com/sitemap.xml`

**Google Analytics 4:**
1. Create property at https://analytics.google.com
2. Add tracking code to `app/layout.tsx`
3. Set up conversion events

**Google AdSense:**
1. Apply at https://adsense.google.com
2. Add site
3. Wait for approval (1-2 weeks)
4. Add ad units after approval
