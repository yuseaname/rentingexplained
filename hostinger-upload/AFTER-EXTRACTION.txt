?  ?? TROUBLESHOOTING                                                       ?
??????????????????????????????????????????????????????????????????????????????

Problem: App status shows "Stopped" or "Error"
Solution:
  ? Click "View Logs" in hPanel to see error
  ? Verify startup file is exactly: server.js
  ? Check Node version is 18.x or 20.x
  ? Try clicking "Restart"

Problem: Build failed or didn't run
Solution:
  ? Click the "Terminal" button in hPanel
  ? Or SSH: ssh username@yourdomain.com
  ? Run these commands:
    cd domains/yourdomain.com
    npm install --production
    npm run build
  ? Then restart app in hPanel

Problem: Page shows 404 or routing doesn't work
Solution:
  ? Verify .htaccess file exists
  ? In File Manager, check for .htaccess
  ? If missing, rename .htaccess-hostinger to .htaccess
  ? Restart the application

Problem: Images don't load
Solution:
  ? SSH to server or use hPanel Terminal
  ? Run: chmod -R 755 public/
  ? Restart application

Problem: Port conflict or port not assigned
Solution:
  ? Stop the application
  ? Wait 30 seconds
  ? Start it again
  ? Port should be auto-assigned

?  ?? UPDATING YOUR SITE (FUTURE)                                           ?

To update your site later:

Method 1: Quick Update
  1. Make changes locally
  2. Create new ZIP: run create-zip.bat/sh
  3. Upload new ZIP to hPanel
  4. Extract (overwrite existing files)
  5. Restart Node.js app

Method 2: Individual Files
  1. Upload changed files via File Manager
  2. If code changed: SSH and run npm run build
  3. Restart app in hPanel

Method 3: Git (Best for frequent updates)
  1. Set up Git repository
  2. SSH to server
  3. git pull && npm run build
  4. Restart app
