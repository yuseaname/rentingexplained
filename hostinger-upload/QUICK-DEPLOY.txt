??????????????????????????????????????????????????????????????????????????????
?  STEP 1: CREATE ZIP FILE                                                  ?
??????????????????????????????????????????????????????????????????????????????

Windows:  Double-click "create-zip.bat"
Mac/Linux: Run "bash create-zip.sh"

This creates: rentingexplained.zip (~400 KB)

??????????????????????????????????????????????????????????????????????????????
?  STEP 2: UPLOAD TO HOSTINGER                                              ?
??????????????????????????????????????????????????????????????????????????????

1. Go to: https://hpanel.hostinger.com
2. Click: Files ? File Manager
3. Navigate to: domains/yourdomain.com/
4. Click: Upload (select rentingexplained.zip)
5. Right-click ZIP ? Extract Here
6. Wait for extraction to complete

??????????????????????????????????????????????????????????????????????????????
?  STEP 3: CONFIGURE & START                                                ?
??????????????????????????????????????????????????????????????????????????????

1. In hPanel, go to: Advanced ? Node.js
2. Click: Create Application
3. Set these:
   ? Node version: 18.x
   ? Startup file: server.js
   ? App root: domains/yourdomain.com
4. Add environment variables:
   ? NODE_ENV = production
   ? NEXT_PUBLIC_SITE_URL = https://yourdomain.com
5. Click: Start

??????????????????????????????????????????????????????????????????????????????
?  DONE! Visit https://yourdomain.com                                       ?
??????????????????????????????????????????????????????????????????????????????

?? Total Time: 10-15 minutes

Need help? See EASY-DEPLOY-GUIDE.txt for detailed instructions.

Troubleshooting:
- App won't start? Check logs in hPanel ? Node.js ? View Logs
- Need to rebuild? SSH and run: npm install && npm run build
