================================================================================
  UPLOAD DESTINATION
================================================================================

Upload TO:    domains/yourdomain.com/
Via:          FTP (ftp.yourdomain.com) or SFTP (port 22)
Username:     Your Hostinger username
Password:     Your Hostinger password

================================================================================
  QUICK START
================================================================================

1. Connect via FTP/SFTP
2. Navigate to domains/yourdomain.com/
3. Upload ALL files and folders from this directory
4. SSH to server: ssh username@yourdomain.com
5. Navigate: cd domains/yourdomain.com
6. Install: npm install --production
7. Build: npm run build
8. Configure Node.js app in hPanel
9. Set startup file: server.js
10. Start the application

================================================================================
  CRITICAL FILES
================================================================================

Must have these files for the app to work:

  ? server.js          - Without this, app won't start!
  ? .htaccess          - Without this, routing won't work!
  ? package.json       - Without this, can't install dependencies!
  ? next.config.js     - Without this, Next.js won't configure!

After upload, verify with:
  ls -la server.js
  ls -la .htaccess
  ls -la package.json

================================================================================
  WHAT HAPPENS AFTER UPLOAD
================================================================================

ON SERVER:
1. npm install --production  ? Creates node_modules/ (~200-300 MB)
2. npm run build             ? Creates .next/ (~50-100 MB)
3. Start app in hPanel       ? Runs on assigned port (3000-4000)
